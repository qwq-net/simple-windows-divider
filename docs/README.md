# ドキュメント索引

windows-divider は、Windows 11 向けの常駐型ウィンドウ管理ユーティリティです。標準スナップの無効化、矢印ホットキーによるグリッド配置、アプリ別のウィンドウ位置の自動復元を提供します。ゲームと併用してもアンチチートに弾かれないよう、文書化済みのユーザーモード Win32 API だけで構成しています。

このフォルダには、利用と開発のための詳細ドキュメントを置いています。概要だけ知りたい場合はリポジトリ直下の [README](../README.md) を参照してください。

| [architecture.md](architecture.md) | 全体の設計。純ロジックと Win32 副作用層の分離、モジュール構成、メッセージループ。 |
| [features.md](features.md) | 3 つの機能（スナップ無効化・グリッド配置・自動復元）の仕様と使い方、動作要件、既知の制約。 |
| [anti-cheat.md](anti-cheat.md) | アンチチート安全性の方針。使う API と使わない API、介入可否の判定。 |
| [configuration.md](configuration.md) | 設定ファイル `config.toml` の各項目とホットキー文字列の書式。 |
| [build-and-test.md](build-and-test.md) | ビルド手順、テスト、実機での確認手順。 |
